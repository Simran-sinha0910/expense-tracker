# Expense Tracker (Vite + React)

This is a React port of your existing HTML/CSS/JS expense tracker using Vite.

## Scripts
- npm install
- npm run dev
- npm run build
- npm run preview

## Structure
- src/pages: Login, Register, Dashboard, Profile, AddExpense
- src/components: Header, Footer, ExpenseCard, ProtectedRoute
- src/context: AuthContext
- src/api: placeholder API clients
- public: put images here (ex1.jpg, ex2.jpg, ex3.jpg)

Copy your images from your existing frontend into `public/` and update code if names differ.
